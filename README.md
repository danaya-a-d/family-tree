# Family Tree (React + TypeScript)

Учебный проект визуализации семейного древа.

## 🚀 Стек
React, TypeScript, ESLint, Prettier.

## 📦 Установка и запуск
```bash
npm i
npm run dev       # старт дев-сервера
npm run build     # сборка прод-версии
npm run preview   # предпросмотр сборки (если есть)
npm run lint      # линт (если настроен)
npm run typecheck # проверка типов
